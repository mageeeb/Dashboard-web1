CHAPITRE 01 Les sélecteurs

+

jQuery te permet de cibler des éléments du DOM avec la même syntaxe que CSS. Le tout passe par $() — la fonction phare de la bibliothèque.

// $(".classe") — par classe $(".item").css("background", "#ffd23f");
Résultat
  • Item 1
  • Item 2
  • Item 3
// :eq() — sélectionne par index $(".item").eq(1).addClass("highlight");
Résultat — seul le 2e item
  • Premier
  • Deuxième
  • Troisième
À retenir : $(this) dans un événement représente l'élément qui a déclenché l'action. Indispensable pour cibler "celui sur lequel je viens de cliquer".

CHAPITRE 02 Les événements

+

Les événements connectent l'interface à la logique. Click, hover, keypress, submit… jQuery les unifie avec .on() et ses raccourcis.

// .click() — réagit au clic $("#btn").click(function() { $(this).text("Cliqué !"); });
Résultat — clique-moi
// .hover() — souris dessus / sortie $(".box").hover( function() { $(this).css("background", "#1aa179"); }, function() { $(this).css("background", "#ff4d2e"); } );
Survole le carré
// .on("keyup") — frappe au clavier $("#in").on("keyup", function() { $("#out").text($(this).val()); });
Tape, l'écho s'affiche
[en attente...]

CHAPITRE 03 CSS & classes

+

jQuery facilite la manipulation du style : .css(), .addClass(), .removeClass(), .toggleClass().

// .toggleClass() — ajoute ou retire $("#box").toggleClass("highlight");
Clique pour basculer
// .css() avec un objet $("#box").css({ "background": "#2d5fff", "transform": "rotate(15deg)", "border-radius": "50%" });
Plusieurs propriétés d'un coup

CHAPITRE 04 Effets fadeIn / slideUp / toggle

+

jQuery propose des animations prêtes à l'emploi : .fadeIn(), .fadeOut(), .slideUp(), .slideDown(), .toggle().

// .fadeToggle() — apparaît/disparaît $("#box").fadeToggle(600);
Animation d'opacité
// .slideToggle() — déroule/replie $("#box").slideToggle(500);
Animation de hauteur
Astuce : les durées s'expriment en millisecondes (400 par défaut). Tu peux aussi passer "slow" ou "fast".

CHAPITRE 05 .animate() — animations custom

+

Pour des animations sur mesure, .animate() interpole n'importe quelle propriété CSS numérique.

// .animate() — déplace en X $("#box").animate({ "margin-left": "200px", "opacity": 0.5 }, 1000);
Déplacement animé

CHAPITRE 06 Manipulation du DOM

+

.append(), .prepend(), .remove(), .html(), .text() : le toolkit pour modifier la structure de la page à la volée.

// .append() — ajoute à la fin $("#list").append( "<li>Nouvel item</li>" );
Ajout dynamique
  • Item initial
// .html() vs .text() $("#out").html("<strong>Hello</strong>"); // .text() afficherait les balises littéralement
Différence html / text

CHAPITRE 07 Le chaining

+

Chaque méthode jQuery retourne l'objet jQuery, ce qui permet d'enchaîner les appels. Plus lisible, plus concis.

// Chaîne de 4 actions en un seul appel $("#box") .css("background", "#ffd23f") .animate({ "margin-left": "100px" }, 500) .animate({ "margin-left": "0" }, 500) .css("background", "#ff4d2e");
4 actions enchaînées
Pourquoi ? Les méthodes "setters" (qui modifient) retournent this. Les méthodes "getters" (qui lisent, comme .val(), .text() sans argument) cassent la chaîne car elles retournent une valeur.

CHAPITRE 08 .each() et data-attributes

+

.each() itère sur chaque élément d'une sélection. .data() lit les attributs data-* en HTML. La combinaison est la base des composants dynamiques.

// .each() — boucle sur la sélection $(".item").each(function(i) { $(this).text( (i + 1) + ". " + $(this).data("label") ); });
Lit data-label et numérote

  • --
  • --
Pattern fréquent : exactement ce que fait notre navbar ! Chaque <li> porte un data-page="...", et le clic lit cette donnée pour savoir quelle section afficher.
